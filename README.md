## Minimax Tic-Tac-Toe AI
本项目基于 C++98 实现了一个使用 Minimax 算法的井字棋 AI。

引擎内置了自定义的启发式评估函数，能够精准区分活二、死二等局部棋型以优化搜索策略。

在纯文本控制台交互下，该 AI 具备完美的理论博弈能力，玩家无法取得胜利。

## 使用方法
启动后，玩家执 'X'，AI 执 'O'。根据终端打印的棋盘布局，输入 1-9 的数字在对应位置落子即可。

确保本地环境支持 C++98 标准，执行以下命令进行编译与运行：
~~~
g++ main.cpp -o minimax -std=c++98./minimax
